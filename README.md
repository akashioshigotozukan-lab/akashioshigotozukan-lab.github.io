# あかし おしごとずかん

明石市のこどもたちが、地元にどんなおしごとがあるのかを検索できるサイトです。

**https://akashioshigotozukan-lab.github.io/**

## これはなに

明石市内にある法人 7,395件を、こども向けの12のカテゴリ（つくる／たてる／うみの／なおす・たすける など）に分けて検索できるようにしたものです。会社名・フリガナ・町名で検索でき、住所をクリックするとGoogleマップが開きます。

## データについて

- 出典：[国税庁 法人番号公表サイト](https://www.houjin-bangou.nta.go.jp/) 全件データ（2026年6月30日時点）
- 対象：兵庫県明石市に所在する法人 7,395件

法人番号データには業種の情報が含まれていないため、カテゴリは**会社名から自動で推測**しています。そのため次の制約があります。

- 約59%（4,378件）は「そのほか」になります。カタカナの社名や人名の会社は、名前から業種を判断できないためです
- 分類が実際の事業内容と異なる場合があります
- **個人事業主は含まれません。** 法人番号は法人にしか付与されないため、個人商店・農家・漁業者などは掲載されていません
- 住所は登記上の所在地です。実際の店舗や工場の場所とは異なることがあります

## 構成

`index.html` の1ファイルのみです。データを埋め込んであるため、外部ファイルもライブラリも使用していません。ファイルをダウンロードして開けば、オフラインでも動作します。

データの生成スクリプトは別途管理しています。
